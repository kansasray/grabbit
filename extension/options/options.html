Audio Capture (Netflix, Apple TV+, DRM content)

The popup's Audio Capture section records via OBS instead of yt-dlp — works for DRM-protected sites where direct download is impossible. Requires one-time setup:

  1. Install OBS 30+ (32+ recommended)
  2. Grant Screen Recording permission to OBS in macOS Settings
  3. In OBS: Sources + → macOS Audio Capture → pick Google Chrome
  4. OBS Tools → WebSocket Server Settings → Enable + Generate Password → save password to ~/.config/grabbit/obs_password
  5. Output → Recording → Custom FFmpeg, format mp3, file path /tmp/grabbit_recordings/output.mp3
  6. Restart the backend (launchd will reload start.sh and pick up the password)

Recordings are saved to ~/Downloads/grabbit/audio/. While recording, don't play other audio inside the same Chrome window — it'll be captured too. Other apps (Spotify, Slack, etc.) are unaffected.
